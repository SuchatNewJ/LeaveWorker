🔗 เชื่อมต่อ Google Sheet
ระบบนี้เก็บข้อมูลลงใน Google Sheet โดยตรง
ต้องตั้งค่าครั้งแรกครั้งเดียว จากนั้นจะจำ URL ไว้ให้อัตโนมัติ
ขั้นตอนการตั้งค่า:
  1. เปิด Google Sheet ใหม่หรืออันที่มีอยู่
  2. คลิก Extensions → Apps Script
  3. วางโค้ดจากไฟล์ Code.gs ที่ได้รับ
  4. รันฟังก์ชัน setup() ครั้งแรก (กด ▶)
  5. คลิก Deploy → New deployment
  6. เลือก Web app, ตั้ง Who has access เป็น Anyone
  7. คลิก Deploy แล้วคัดลอก Web app URL
  8. วาง URL ด้านล่าง แล้วกดเชื่อมต่อ
